Standalone projects

Since all of your files are in the /src/project/ folder, upgrading is easy.

  1. Download and unzip the latest release
    • You can also download specific releases
  2. Replace everything in /src/ except your project folder
    • It's recommended that you delete the /dist/ folder to clear out any old files. All necessary files and folders will be recreated.
  3. On the command line, run npm install to make sure you have the latest Node modules for the release
  4. Use Grunt to build your project as usual
