Core UI » Grid

Core UI's grid was created for developers who need a simple but solid grid system. Core UI's grid easily adapts to any size of layout and it supports a combination of fixed and fluid columns. It is also customizable via SASS.

  • Mobile first
  • Based on rows and columns
  • 12-column default grid
  • Fluid and/or fixed with filler when needed
  • Responsive with three breakpoints (≤480px, ≤768px and ≥769px)
  • Gutters
  • Push columns
  • Hide columns
  • Mixins for customizing columns, gutters, etc.
